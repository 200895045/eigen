The show pages display information and works of shows, which are exhibitions at either galleries or fairs. They are intended to learn about shows, how to visit them, and discover art by themes (?).

User stories

Visiting

# e i s
As a visitor, I would like to see important visitor information first, so that I know if I can attend.
As a visitor, I would like to get full opening hours information, so that I can see if attending at other times will work.
Should this still show, e.g. “Friday from 14 - 18” if coming Friday is past the show ending date?

Discovering

# e i s
As a browser, I would like to see only a few artworks if there are many, so that I can wet my appetite but still easily get to other show information.
This section should be collapsed if there 8 or more artworks in the show.

Roles

Visitor
Someone that wants to visit a show.
Browser
Someone that is browsing through the shows and wating to discover shows.

Definitions

Show
